# AgentForge Harness Engineering 技能包

> 模型决定上限，Harness 决定底线。

一键安装 AgentForge 框架的完整 Harness Engineering 方法论：18条铁律 + 8智能体Bug修复技能 + 全链路验证。

## 快速安装

```bash
# 克隆
git clone https://github.com/paskaa/agentforge-harness-skill.git

# 安装
bash agentforge-harness-skill/scripts/install.sh
```

## 安装内容

| 组件 | 说明 |
|---|---|
| `rules/IRON_LAWS.md` | 18条铁律（不可违反） |
| `skills/agentforge-*` | 6个Bug修复技能 |
| `skills/harness-engineering` | Harness Engineering方法论 |
| `agents/` | 8个智能体定义 |

## 铁律概览

1. Bug状态管理
2. 修复流程
3. 全链路6环分析
4. 状态值一致性
5. 影响面分析
6. 逆向流程验证
7. 全链路验证
8. 池/统计表同步
9. 统计变更验证
10. 禁止删除源文件
11. 禁止改方法签名
12. 搜索代码路径
13. 数据库铁律
14. 测试铁律
15. 归档铁律
16. 禅道交互
17. 质量门禁
18. 禁止硬编码默认值

## 适用场景

- Bug修复（前端/后端/全栈）
- 代码审查
- 新功能开发
- 任何需要遵循Harness Engineering方法论的任务
